## The Four Mandatory Repositories

AFT requires four Git repos (can be empty initially):

- **aft-account-request** — account HCL files
- **aft-global-customisations** — all accounts
- **aft-account-customisations** — per-account
- **aft-account-provisioning-customisations** — pre-baseline

---

## Pre-flight Check

- Control Tower Landing Zone active
- Three account IDs to hand
- Four Git repos created
- Git integration configured
- IAM Identity Centre enabled
- Terraform backend ready

All green → deploy AFT.
